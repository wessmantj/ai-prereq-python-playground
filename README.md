# SELF NOTES
 
## What is venv?

    - Virtual Environment or venv is a standard built-in Python tool used for creating isolated virtual environments for installations and projects.

### Why use venv?

    - Virtual environments allow for isolation of project specific packages in a directory, ability to use different versions of the same library for different projects, and avoids installing unecessary packages globally on device.
